# 资产收益助手（基金/期货）

> Chrome 浏览器扩展 · 本地运行 · 无需登录 · 数据存储在本地

一款专为个人投资者设计的基金/期货持仓管理工具，支持实时行情同步、自动日结算、OCR 批量录入等功能，所有数据保存在本地，不上传任何服务器。

---

## 功能概览

### 📊 持仓总览
- 以表格形式展示所有持仓的代码、名称、分组、持仓金额、份额、净值、昨日收益、今日估算收益、累计收益
- 支持按任意列排序（点击列头切换升/降序，带箭头指示）
- 支持分组筛选（下拉菜单快速切换"全部/自定义分组"）
- 汇总行实时显示总持仓金额、昨日总收益、今日估算总收益、累计总收益

### 🔄 实时行情同步
- 自动拉取场外基金净值（天天基金主接口 + 东方财富备用接口）
- 支持场内基金（ETF/LOF）和期货品种（通过新浪行情代理）
- 每个基金请求带 8 秒超时保护，超时自动降级显示，不阻塞整体加载
- 点击顶部刷新按钮可随时手动同步

### 💰 日结算
- **自动结算**：每次启动时若检测到净值变化且今日未结算，自动完成结算并 Toast 提示
- **手动结算**：通过悬浮菜单触发，弹窗二次确认后执行
- **撤销结算**：结算前自动备份数据，当天内可一键撤销恢复到结算前状态
- 结算核心逻辑：按持有份额 × 净值变化计算实际收益，写入累计收益

### ➕ 添加 / 编辑资产
- 输入资产代码后自动拉取最新净值，金额与份额联动计算
- 支持手动填写持仓金额、份额、累计盈亏、昨日收益、所属分组
- 直接点击表格中的金额、份额、累计收益单元格可内联编辑并自动保存

### 📈 加仓 / 减仓
- 点击任意行的 ⚙️ 图标打开操作弹窗
- **加仓**：输入买入金额和手续费率，自动计算预计确认日期（T+1 或 T+2）
- **减仓**：输入卖出份额，支持"全部"一键填满
- 提供 15:00 前/后切换器，精准匹配确认日期
- 待确认记录在下次加载时根据净值自动确认份额，可在"交易记录"中查看和撤销

### 🗂 分组管理
- 任意资产可归属自定义分组（默认/已撤回/…）
- 清空持仓时可选择同时将分组改为"已撤回"
- 批量操作：多选后统一修改分组

### ✅ 多选批量操作
- 点击行切换选中状态，Shift+点击 范围选
- 选中后顶部状态栏显示已选数量，支持一键取消
- 批量操作（通过悬浮菜单）：
  - 全选 / 取消全选
  - 批量重算份额（按当前金额 ÷ 最新净值）
  - 批量修改分组
  - 批量清空持仓
  - 批量删除资产

### 📷 OCR 批量录入
- 截图上传基金 App 页面，自动识别基金名称、持仓金额、昨日收益、持有收益
- 支持列表页（多基金）和详情页（单基金含代码）双模式解析
- 识别后自动反查 6 位基金代码（调用东方财富搜索接口）
- 结果展示为可编辑表格，手动核对后批量保存
- 也支持直接粘贴文本（格式：`代码 金额`，每行一条）

### 💾 导入 / 导出
- 导出为 JSON 文件（含持仓、累计收益、昨日收益等全量数据），文件名带时间戳
- 导入时自动做格式校验和数据迁移（兼容旧版字段结构）
- 导入前需二次确认，防止误操作覆盖当前数据

## 使用方式

1. 在 Chrome 扩展管理页面（`chrome://extensions`）开启开发者模式
2. 点击"加载已解压的扩展程序"，选择本项目文件夹
3. 点击浏览器工具栏的扩展图标即可打开弹窗
4. 点击右上角 ⛶ 图标可在独立标签页全屏显示

---

## 数据说明

- 所有持仓数据通过 `chrome.storage.local` 保存在本地浏览器，卸载扩展后数据会清除
- 建议定期使用"导出"功能备份数据
- 行情数据来源：天天基金（场外）、东方财富（备用）、新浪行情（场内/期货）

---

## 支持的资产类型

| 类型 | 代码格式 | 示例 |
|------|---------|------|
| 场外基金 | 6 位纯数字 | `005827`、`110011` |
| 场内 ETF / LOF | 6 位纯数字（5 开头或 1/3 开头） | `510300`、`159915` |
| 期货 | 字母+数字 | `RB2501`、`AU2506` |
